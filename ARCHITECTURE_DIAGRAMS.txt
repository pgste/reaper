DATA STORE ARCHITECTURE (ABAC/ReBAC)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

DataStore (data/store.rs)
├── StringInterner (Arc)
│   └── ~60% memory savings via deduplication
│
├── Primary Index: EntityId -> Arc<Entity>
│   ├── ID lookup: 20-50ns
│   └── DashMap for lock-free access
│
├── Type Index: EntityType -> HashSet<EntityId>
│   └── Type lookup: 100-200ns
│
├── Attribute Index: (AttrKey, AttrValue) -> HashSet<EntityId>
│   └── Attribute lookup: 100-300ns (indexed)
│
└── Composite Index: (Type, AttrKey, AttrValue) -> HashSet<EntityId>
    └── Complex queries: optimized
